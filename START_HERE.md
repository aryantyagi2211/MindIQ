## ⚡ Quick Start (5 Minutes)

### 1️⃣ Get Groq API Key
- Go to: https://console.groq.com
- Sign up and create an API key
- Copy it (starts with `gsk_...`)

### 2️⃣ Add to .env File
Open `mind-ranker-main/.env` and replace:
```env
GROQ_API_KEY="your_groq_api_key_here"
```
With your actual key:
```env
GROQ_API_KEY="gsk_your_actual_key_here"
```

### 3️⃣ Push to GitHub
```bash
cd mind-ranker-main
git add .
git commit -m "Add AI Paper Assessment"
git push origin main
```

### 4️⃣ Wait for Lovable
- Lovable auto-deploys in 2-3 minutes
- Check your Lovable dashboard

### 5️⃣ Test It!
```bash
npm run dev
# Go to: http://localhost:5173/test/setup
```
