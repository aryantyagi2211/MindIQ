## 🐛 Troubleshooting

### If Questions Don't Generate

**Check 1: Groq API Key**
```bash
# Verify key is in .env
cat .env | grep GROQ_API_KEY
```

**Check 2: Browser Console**
- Open DevTools (F12)
- Check Console tab for errors
- Look for network errors

**Check 3: Function Logs**
- Go to Lovable Dashboard
- Check function logs for errors
- Verify Groq API is responding

### If You See Errors

**"Failed to generate questions"**
- Check Groq API key is correct
- Verify Groq account has credits
- Check function logs in Lovable

**"Functions not found (404)"**
- Wait 2-3 minutes for deployment
- Refresh browser
- Check Lovable deployment status

**CORS Errors**
- Functions include CORS headers
- Clear browser cache
- Try incognito mode
